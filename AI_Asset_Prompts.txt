Three categories of evidence:

  [AI-GENERATE]   Physical objects / photographs / scene items
                  → Need a real image. Generate on transparent background (PNG).
                  → These are shown in the EvidenceDetailPanel's viewer area.

  [UNITY UI]      Documents, logs, digital readouts, text records
                  → Can be rendered entirely in Unity as styled text panels.
                  → No image needed. The description text IS the evidence.
                  → Use a generic "paper document" or "terminal screen" sprite
                     (one reusable sprite for all documents across all cases).

  [REUSE]         Generic object (receipt, keycard, badge) that can be shared
                  across multiple cases with a different UI text overlay.

BACKGROUNDS (full scenes, JPG is fine):
  Tool: Leonardo AI (app.leonardo.ai)
  Model: Leonardo Diffusion XL or Kino XL
  Dimensions: 1080 × 1920 (portrait, matches our CanvasScaler)
  Steps: 30+, Guidance 7-8
  Lock style: Use "Image to Image" with the noir office photo as style reference
              to keep consistent visual tone across all cases (30-40% influence)

EVIDENCE OBJECTS on transparent background (PNG):
  Tool: Stable Diffusion (locally) or Adobe Firefly (Generative Fill trick)
  Method A — Stable Diffusion:
    Model: Realistic Vision 5.0 or DreamShaper
    Prompt suffix always add: "isolated on pure white background, product photography"
    Then in Photoshop/GIMP: Select by color (white) → Delete → Export PNG
  Method B — Adobe Firefly / Photoshop:
    Generate the object, use "Remove Background" (one click in Photoshop 2024+)
  Method C — remove.bg (free tier):
    Generate anywhere, upload to remove.bg, download as transparent PNG

CCTV / SECURITY STILLS (no transparency):
  Tool: Midjourney or Leonardo
  Prompt style: "--style raw --ar 4:3" in Midjourney
  Add: "security camera footage, 1990s CCTV, grainy, black and white, scanlines"
