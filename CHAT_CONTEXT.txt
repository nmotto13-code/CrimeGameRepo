PROJECT OVERVIEW
----------------
Unity 6 (6000.x LTS) iOS-first mobile mystery game.
Primary working directory: C:\Users\blued\OneDrive\Desktop\CrimeGame

Two editor menu commands drive everything:
  1. Casebook → Generate Starter Cases   (creates/updates ScriptableObjects)
  2. Casebook → Build Scene              (rebuilds CaseScene.unity from scratch)
Then Press Play to test.

ARCHITECTURE SUMMARY
--------------------
- CaseData ScriptableObject → EvidenceData / ClaimData / HotspotData
- CaseLoader.LoadCase() wires everything at runtime
- EvidenceDiscoverySystem tracks found evidence, populates Evidence tab
- MagnifyingGlass (on ScenePanel) detects hotspot proximity via screen-distance
- HotspotController.TriggerDiscovery() → registers evidence + opens detail panel
- TabController manages 4 tabs: Brief(0) Scene(1) Evidence(2) Solve(3)
- ContradictionEvaluator handles claim selection and correct/wrong evaluation
